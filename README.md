# 🕷️ Web Scraper

A professional, flexible Python web scraping tool that extracts data from *any website* — headings, links, images, tables, or custom CSS selectors — and exports results to *CSV, JSON, and Excel*.

---

## ✨ Features

- *6 scraping modes* — titles, links, images, tables, custom selector, or all at once
- *Custom CSS selector* — target any element on any page
- *Multi-format export* — CSV, JSON, and Excel (.xlsx) with one run
- *Retry logic* — auto-retries failed requests up to 3 times
- *Duplicate-safe links* — no repeated URLs in results
- *Full logging* — timestamped .log file after every run
- *Realistic headers* — mimics a real browser to avoid blocks
